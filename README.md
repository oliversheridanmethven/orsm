## Setting up the repo

### Dependencies

The project relies on a few dependencies, the most notable of 
which include:

- GCC (with C23 and C++23 support).
- CMake.
- GLIB (for logging).
- Argp (for the CLI).
- Criterion (for testing).

### Build

To build the project:
```bash
cd build
cmake .. 
make 
make test
```
